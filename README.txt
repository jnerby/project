DESCRIPTION
	watchlist is a web-based application that allows users to start and join film clubs. Club members can view film recommendations and search for films to add to their club’s watchlist. Members can schedule dates to view films as a club and rate films. An individual member’s recommendations adjust based on a member’s highly rated films (8/10 or higher).

TECH STACK
Languages: Python, Javascript, HTML, CSS, SQL
Frameworks & Libraries: Flask, React, Jinja, Boostrap
APIs: TMDB, Twilio

	This app uses React to render the following pages: watchlist, Upcoming, and Log. All film data and posters come from TMDB’s API. This app also uses Twilio to send messages to club members when a film viewing is scheduled or a scheduled viewing is canceled.
